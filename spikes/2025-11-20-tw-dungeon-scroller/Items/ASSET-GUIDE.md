## Download & Installation

### Schritt 1: Assets herunterladen

1. Gehe zu den Links oben
2. Klicke auf "Download" (kostenlos)
3. Entpacke die ZIP-Dateien

### Schritt 2: Icons auswählen

Für unsere **Common Items** brauchen wir absichtlich **schlechte/einfache** Icons:

| Item | Gesuchtes Icon | Pack |
|------|----------------|------|
| Verbeulter Topfhelm | Einfacher/hässlicher Helm, am besten grau/braun | Clockwork Raven |
| Geflickte Ledertunika | Einfache Leder-Brustplatte, braun | Clockwork Raven |
| Rostiges Kurzschwert | Einfaches/altes Schwert, grau/braun | Willibab |
| Holzbrett mit Griff | Einfacher/kaputter Schild, Holz-Optik | Clockwork Raven |
| Ausgebeulte Stoffhose | Einfache Hose/Beinschienen, braun | Clockwork Raven |
| Abgelaufene Lederstiefel | Einfache Stiefel, abgenutzt | Clockwork Raven |

### Schritt 3: Icons umbenennen und kopieren

Benenne die ausgewählten Icons um und kopiere sie nach:
```
next-app/public/Assets/Items/Icons/
```

Dateinamen:
- `equipment_helm_topf.png`
- `equipment_brustplatte_geflickt.png`
- `equipment_schwert_rostig.png`
- `equipment_schild_brett.png`
- `equipment_hose_ausgebeult.png`
- `equipment_schuhe_abgelaufen.png`

### Schritt 4: (Optional) Sprites für World-Drop

Für 16x16 World-Sprites (wenn Items auf dem Boden liegen):
```
next-app/public/Assets/Items/Sprites/
```

## Tipps für Common Items

Common Items sollen **schlecht und langweilig** aussehen:

- Wähle die **einfachsten/hässlichsten** Icons aus dem Pack
- Bevorzuge **graue, braune, beige** Farben
- Keine leuchtenden/magischen Effekte
- Keine goldenen Verzierungen
- Je kaputter/einfacher, desto besser

Für **Rare/Epic/Legendary** Items später:
- Leuchtende Farben (grün, blau, lila, gold)
- Magische Effekte/Glühen
- Aufwändigere Designs
- Verzierungen und Details
